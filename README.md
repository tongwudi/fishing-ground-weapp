## fishing-ground-weapp

### 安装与使用

- 下载代码

```bash
git clone https://github.com/tongwudi/fishing-ground-weapp.git
```

- 安装

```bash
cd fishing-ground-weapp

npm install
```

- 运行

```bash
工具 -> 构建 npm
```

---

### 目录结构

```
├─activity: 活动
├─fish: 鱼种
```
